# FamilyVault 📁

A simple, secure document safe built for Indian families.

## The Problem
My family was losing important documents — Aadhaar cards, land papers,
medical records — scattered across WhatsApp groups that delete files
over time. I created a WhatsApp group as a temporary fix, but files
still got lost. So I built FamilyVault.

## What it does
- Upload documents with a name and category
- Join a family group with a simple code — no complex signup
- Search any document instantly by name or category
- Works on mobile browser — no app download needed for elders

## Tech Stack
- Backend: Spring Boot + JPA + MySQL
- Frontend: React + Tailwind CSS (in progress)
